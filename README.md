🏠 House Price Prediction - Exploratory Data Analysis (EDA)
📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a house price dataset to identify the factors that influence property prices. The analysis includes data cleaning, visualization, correlation analysis, and feature exploration to prepare the dataset for machine learning models.

🎯 Objectives
Analyze the housing dataset to understand its structure.
Clean and preprocess the data.
Identify relationships between different features and house prices.
Detect missing values and outliers.
Visualize trends and patterns using charts.
Prepare the dataset for predictive modeling.
📂 Dataset Features
The dataset includes features such as:

Area (Square Feet)
Number of Bedrooms
Number of Bathrooms
Location Rating
Amenities Score
Property Age
Parking Availability
Furnishing Status
House Price (Target Variable)

📈 Key Insights
House price has a strong positive relationship with area and location rating.
Premium locations generally have higher property prices.
Larger houses tend to be more expensive.
Outliers were identified and analyzed to improve data quality.
